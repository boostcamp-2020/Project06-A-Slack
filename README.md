## 🔎프로젝트 소개

팀 협업도구로 유명한 슬랙을 구현하는 프로젝트 입니다.✨  
기본적인 회원가입부터 사용자별 권한 관리, 채널 관리, 실시간 채팅, 이모지, 멘션 등의 기능을 구현합니다.
